ROOT CAUSE:
The snippet incorrectly assumed `RasterRDD` has a `rasterFeature` property. In RDPro, `RasterRDD[T]` is a type alias for `RDD[ITile[T]]`, so to obtain a `RasterMetadata` instance, you must extract a tile (e.g., via `rasterRDD.first()`) and access its `rasterMetadata`. Furthermore, the returned `Envelope` is a JTS `Envelope`, which uses Java getter methods (`getWidth()` and `getHeight()`) rather than Scala-style property accessors (`width` and `height`).

CORRECT USAGE:
```scala
val metadata = rasterRDD.first().rasterMetadata
val bounds = metadata.envelope
val witness = bounds.getWidth * bounds.getHeight
require(witness > 0, s"degenerate envelope: $bounds")
println("__CHECK__ envelope " + witness)
```
