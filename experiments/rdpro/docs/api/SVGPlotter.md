### Goal
The `getTitle` function extracts and returns a formatted title string for a given feature by interpolating predefined title templates with the feature's attributes.

### Parameters
- `feature` (`IFeature`): The feature from which to extract attribute values for title interpolation. This feature should contain attributes that correspond to the placeholders defined in the `svgTitle` string.

### Input
The caller must provide an `IFeature` instance that contains the necessary attributes for title interpolation. The attributes should match the placeholders defined in the `svgTitle` string, which can include both attribute names and positional references.

### Output
Returns `String` — a formatted title string that is generated by interpolating the `svgTitle` with the values of the attributes from the provided feature.

### Valid Call Patterns
```scala
val plotter = new SVGPlotter
plotter.svgTitle = "record #${id}"
val title = plotter.getTitle(Feature.create(EmptyGeometry.instance, Array("id"), null, Array(15)))
// title will be "record #15"
```
